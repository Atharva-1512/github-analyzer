🚀 GitHub Analyzer

AI-powered tool to analyze GitHub profiles and repositories with actionable insights on tech stack, activity, and code patterns.

📌 Overview

GitHub Analyzer is a modern web application that helps developers, recruiters, and teams quickly understand GitHub profiles and repositories without manual inspection.

It extracts meaningful insights such as:

Tech stack usage
Contribution activity
Repository patterns
Developer behavior

Instead of going through multiple repos manually, this tool gives a centralized, intelligent overview.

🎯 Why This Project?

Manually evaluating GitHub profiles is:

Time-consuming
Inconsistent
Hard to scale

This tool solves that by:

Automating analysis
Standardizing insights
Making decisions faster (especially in hiring)

⚙️ Getting Started
1️⃣ Clone the Repository
git clone https://github.com/Atharva-1512/github-analyzer.git
cd github-analyzer
2️⃣ Install Dependencies
npm install
3️⃣ Setup Environment Variables

Create a .env.local file:

GITHUB_TOKEN=your_github_token_here

⚠️ Token is optional but recommended to avoid rate limits.

4️⃣ Run the Project
npm run dev

Visit:

http://localhost:3000
